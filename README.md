# MySQL-Movies
The following are the skills that I developed as a result of this work:

- Data Retrieval Skills: Ability to use SELECT queries to fetch specific columns or all details, applying filters like year, actor, or language.
- Filtering & Sorting Mastery: Learned how to organize results using ORDER BY, LIMIT, and conditions such as LIKE for pattern matching.
- Aggregation Competence: Gained experience with functions like COUNT() to summarize data by language, actor, director, or year.
- Group Analysis: Developed skills in GROUP BY to categorize data and apply aggregate functions across groups.
- Conditional Group Filtering: Understood the difference between WHERE and HAVING, using HAVING to filter aggregated results (e.g., actors with more than 10 movies).
- Relational Database Thinking: Practiced joining multiple tables (movies, actor, lang) to enrich queries with meaningful attributes instead of IDs.
- Trend Identification: Learned to detect industry patterns such as busiest release years, most prolific actors, or dominant languages.
- Business Insight Generation: Recognized how SQL queries can support decision-making by highlighting top performers and production trends.
- Analytical Workflow Development: Strengthened ability to move from raw data to actionable insights through selection, filtering, grouping, and aggregation.
- Scalability Awareness: Understood that these query techniques scale to larger datasets, making them directly applicable to real-world analytics and reporting.
